Copilot Studio – Luware – Jira Automation

Enterprise AI Automation Case Study — integrating Microsoft Copilot Studio with Luware Nimbus via Direct Line to automate Jira ticket creation and preserve conversation context.

Problem

Enterprise contact center teams were handling high call volumes through Luware Nimbus with no AI assistance and manual Jira ticket creation.

  • High agent handling time
  • Inconsistent ticket data
  • Loss of conversation context
  • Limited automation and auditability

My Role

Solution Architect & Technical Product Owner

Owned end-to-end design and delivery, including:

  • Overall solution architecture
  • Copilot Studio conversation design
  • Direct Line integration strategy
  • Jira workflow mapping
  • Security, governance, and error handling

Solution Overview

Designed and delivered an AI-driven Copilot Studio solution integrated with Luware Nimbus via Direct Line.

The solution enables:

  • AI-driven conversations with callers
  • Seamless bot-to-agent handover
  • Automated Jira ticket creation with full context
  • Traceability using Conversation ID and caller identifiers

Architecture Components

  • Microsoft Copilot Studio — Conversational AI
  • Direct Line API — Real-time message exchange
  • Power Automate — Workflow orchestration
  • Jira Service Management — Ticket creation
  • Luware Nimbus — Contact center platform
  • Microsoft Teams — Agent handover

How the Flow Works

  1. Caller interacts with Copilot chatbot
  2. Copilot identifies intent and gathers context
  3. Messages are exchanged via Direct Line
  4. Power Automate creates a Jira ticket
  5. Conversation ID and AI summary are stored
  6. Agent receives ticket with full context

Challenges & Architectural Decisions

Challenges

  • Maintaining conversation context across systems
  • Avoiding duplicate Jira ticket creation
  • Secure handling of authentication tokens
  • Managing conditional handover logic

Key Decisions

  • Used Direct Line for real-time control
  • Persisted Conversation ID for traceability
  • Implemented conditional logic in Power Automate
  • Separated AI logic from ticket-creation workflows

Impact

  • 70% reduction in manual ticket creation effort
  • Improved response time and auditability
  • Scalable foundation for future AI enhancements
